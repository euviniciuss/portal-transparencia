## 💡 Portal de Transparência
O Portal de Transparência é uma plataforma desenvolvida para facilitar o acesso do cidadão às informações sobre a gestão pública, garantindo maior fiscalização e controle social.

## ℹ Sobre o projeto
Este projeto foi desenvolvido durante o **Hackathon da Transparência Maranhense**, promovido pela [Secretaria de Estado da Transparência e Controle do Maranhão (STC)](https://www.instagram.com/stcmaranhao/?hl=pt) em parceria com a [Secretaria de Estado de Ciência, Tecnologia e Inovação do Maranhão (SECTI)](https://www.secti.ma.gov.br/).

O objetivo principal foi modernizar a visualização de dados públicos, oferecendo uma interface intuitiva, rápida e responsiva. Através dele, é possível acompanhar indicadores, repasses municipais e outras informações essenciais de forma clara e transparente, utilizando as melhores práticas de desenvolvimento web moderno.

## ▶️ Como rodar
  ### **Pré-requisitos**
  - É **necessário** ter instalado o **[Node.js](https://nodejs.org/en)** de forma global na sua máquina.
  
  ### **Versão das tecnologias utilizadas**
    - next: 16.2.4
    - react: 19.2.4
    - typescript: ^5
    - tailwindcss: ^4
    - biome: ^2.4.13
  
 ### **Passo a passo para rodar a aplicação**  
1. Faça o clone do repositório:

```sh
  $ git clone https://github.com/euviniciuss/portal-transparencia
```

2. Executando a Aplicação:

```sh
  $ cd stc
  $ npm install
  $ npm run dev
```
